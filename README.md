Perfumes Full Stack
Proyecto CRUD de perfumes usando React, Nodejs, Express y MongoDB Atlas.

Tecnologías
- React
- Vite
- Express
- MongoDB Atlas
- Mongoose
- Tailwind CSS
- DaisyUI

 Funcionalidades
- Crear perfumes
- Listar perfumes
- Editar perfumes
- Eliminar perfumes
- Buscar perfumes

- Durante la conexión del backend con MongoDB Atlas me aparecio un error de resolución DNS relacionado con los registros SRV de MongoDB
  Y use mongoDB Compass para encontrar el problema dns en node.js.
